# Rhythm-Strike

# 提出作品の詳細

## ファイル構成
- Projectファイル: Unity 2023.1.0f1で開く
- README.txt: Projectファイルを開く際に最初に読んでください。

## ソースコードとリソース
- Scripts: 各シーンごとのC#スクリプト
- Shader: シェーダー処理
- Data/Scripts: データベース用スクリプト
- Resources: 音声、テクスチャ、譜面JSON

## 操作
- 決定: Enter / 戻る: Escape / 選択移動: ←→
- ノーツ: D F J K
- 横移動: S L
- ジャンプ: Space
- 一時停止: Escape
- オプション: 現在マウスのみ（キー入力対応予定）
