# 作品名　:　Rhythm Strike!!

### 概要
リズムに合わせてノーツを叩き、敵を倒すアクションリズムゲームです。

## 開発情報
- 開発環境: Unity 2023.1.0f1
- 使用言語: C#
- 開発期間: 2025年9月～2026年2月（約4か月）

## ファイル構成

### プロジェクトファイル
Unityで開くプロジェクト。開くにはUnity 2023.1.0f1が必要です。

### ソースコード
- `Assets/Scripts/` : シーンごとのスクリプト

### シェーダー
- `Assets/Shader/` : シェーダーの処理

### データベース
- `Assets/Datas/Scripts/` : データ管理用スクリプト

## 操作方法

### 全体操作
| 操作 | キー |
|------|------|
| 決定 | Enter |
| 戻る | Escape |
| 選択移動 | ↑↓←→ |

### ゲームプレイシーン
| 操作 | キー |
|------|------|
| ノーツ | D F J K |
| 横移動 | S L |
| 一時停止 | Escape |
